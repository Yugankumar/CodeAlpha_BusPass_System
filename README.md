# Cloud-Based Bus Pass System

## 📌 Project Overview

The Cloud-Based Bus Pass System is a web application developed using Python Flask and SQLite database.

This system allows users to book bus passes online and securely stores ticket details in a database to prevent ticket loss and improve reliability.

The project is designed as part of the CodeAlpha Cloud Computing Internship.

---

## 🚀 Features

- Online Bus Pass Booking
- Secure Ticket Storage
- Dynamic Ticket Generation
- User Registration System
- Flask Web Application
- SQLite Database Integration
- Responsive User Interface
- Cloud Deployment Ready

## ⚙ Installation Steps

### 1. Clone Repository

```bash
git clone YOUR_GITHUB_LINK
```

### 2. Open Project Folder

```bash
cd CodeAlpha_BusPass_System
```

### 3. Create Virtual Environment

```bash
python -m venv venv
```

### 4. Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

### 5. Install Flask

```bash
pip install flask
```

### 6. Run Application

```bash
python app.py
```
